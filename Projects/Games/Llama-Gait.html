Procedural Llama Gait

This Unreal project is a procedural moving animation for a â€‹quadruped. All movement of the body is determined by the placement of the Llama hooves and the current gait the llama is using (walk, pace, etc.)

This primarially relies on the Unreal Control Rig Component. By Solving two independent iks in a particular order we can begin to accomplish correct multi-bone leg movement with no need for a pre-made animation (left images).

The placement of the feet it determined by the current gait cycle, and raycasting on the floor to receive the floor location. This data also determines the rotation of the whole body, the feet, etc. The CR's controls are then set based on these locations and attitive rotations. 

There are many parts that are ready for improvement. For example, the deformations on the model can be significantly improved. The gait cycles system is still dependent on a rigid single cycle rather than being adaptive and still drags along with the body. The Legs should determine more of the characters movement. On top of this, Building a realistic movement system for quadrupeds is very complicated.
