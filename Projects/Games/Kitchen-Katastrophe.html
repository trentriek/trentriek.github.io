Kitchen Katastrophe

My first game I created, this is a Google VRgame built in Unity. Here I learned the very basics of game development and built the entire game. Excluding the game's lag which grew over time, it turned out do be a decent game! The objective was to use two hand to place ingredients into a bowl. But because this was in phone VR, the physics did not work well which made the game a challenge.
