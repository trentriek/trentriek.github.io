Procedural Llama Gait

For my masters Capstone, I attempted to create animal animations that both are reactive to the environment and gameplay input. the objective was that a player could control a four-legged animal model and have it realistically walk within the environment, keeping in mind proper joint movement and gait.

To accomplish this, I abstracted the body movement into three separate sections; leg movement, body movement, and a gait system. I create solvers that procedurally create poses for the leg and body based on the feet locations - Then, the gait system drives the animation by determining said foot location. I attempted this project in the Unreal engine. This was done using a standard Character Blueprint, with a custom animation blueprint. The Leg Solver itself was custom created in C++. 

This was a great project to test complex animation systems, and understanding the core elements of unreal C++ including different custom nodes and module setup. The results are posted to the left.
